TypeScript

TypeScript – це строго типізована мова програмування, яка базується на JavaScript. https://www.typescriptlang.org/

TypeScript

Розвиток TypeScript розпочався наприкінці 2012 року. Хоча він виник в компанії Microsoft, його фактичним творцем є програміст Андерс Хейлсберг, також відомий як творець таких мов, як Delphi, C#. Проте цей проект відразу став розвиватися як OpenSource. І вже з самого початку нова мова швидко поширилась через свою гнучкість і продуктивність. Популярність і актуальність ідей нової мови призвели до того, що деякі з цих ідей пізніше стали частиною нового стандарту JavaScript. Переваги TypeScript були взяті на озброєння творцями ряду поширених і широко використовуваних фреймворків, таких як Angular 2+ та Vue3.

TypeScript

По-перше, слід зазначити, що TypeScript - це строго типізована та компільована мова, що може бути ближчою до програмістів Java, C# і інших строго типізованих мов. На виході компілятор створює той самий JavaScript, який потім виконується браузером. Однак строга типізація зменшує кількість потенційних помилок, які могли б виникнути при розробці на JavaScript.

По-друге, TypeScript реалізує багато концепцій, які є властивими об'єктно-орієнтованим мовам, таким як інкапсуляція, наслідування, поліморфізм та модифікатори доступу тощо.

TypeScript

По-третє, потенціал TypeScript дозволяє швидше і простіше писати великі складні програми, відповідно їх легше підтримувати, розвивати, масштабувати та тестувати, ніж на стандартному JavaScript.

По-четверте, TypeScript розвивається як проект з відкритим кодом і, як і багато інших проектів, розміщений на GitHub. Адреса репозиторію - https://github.com/Microsoft/TypeScript. Крім того, він є крос-платформеним, що означає, що для розробки ми можемо використовувати як Windows, так і MacOS або Linux.

Транспілятор

Транскомпілятор або транспілятор - програма, яка приймає вихідний код, написаний на одній мові програмування і повертає еквівалентний вихідний код на тій самій або іншій мові програмування

Інсталяція та інструменти

  1. Встановити Node.js на ваш компʼютер (nodejs.org)
  2. Відкрити проект у редакторі коду
  3. Ініціалізувати новий проект - npm init -y
  4. Встановити npm пакет TypeScript - npm i -D typescript
  5. Транспіляція ваших файлів - npx (npm) tsc helloworld.ts

tsconfig

Tsconfig.json – файл, вказує на те, що поточна директорія є коренем TypeScript проєкту.

При запуску tsc без вказівки файлу для компіляції використовуватимуться налаштування з файлу tsconfig.json
